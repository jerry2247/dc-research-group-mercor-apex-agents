<description>
When a financial model task requires finding the value of an input parameter that produces a target output metric (such as a specific IRR or yield), implement a Python replica of the model's key calculations and use a numerical search method such as bisection to solve for the parameter.
</description>
<example>
def solve_for_param(target_metric, low=0.0, high=0.2, tol=1e-6):
    for _ in range(50):
        mid = (low + high) / 2
        metric = compute_model_metric(mid)
        if metric < target_metric:
            low = mid
        else:
            high = mid
        if abs(metric - target_metric) < tol: break
    return round(mid, precision)
</example>
</memory_item>

## Formulas, Definitions, and Conventions
<memory_item>
<description>
Midyear convention treats each period's free cash flow as occurring at the midpoint of the year. For annual periods the discount factor for year n therefore uses the fractional exponent (n - 0.5).
</description>
<example>
DF_n = 1 / (1 + r) ** (n - 0.5)
</sample>
</memory_item>

<memory_item>
<description>
The final per-share DCF value is the enterprise value (sum of discounted FCFs plus discounted terminal value) minus net debt, divided by the number of shares, then rounded to the nearest cent.
</description>
<example>
Value_per_share = round( (EV - Net_Debt) / Shares , 2 )
</sample>
</memory_item>
<memory_item>
<description>
In LBO models, MoM (Multiple of Money) is exit equity value divided by sponsor equity invested. IRR is the compound annual return solving for the rate that equates sponsor cash flows, often approximated from MoM as (MoM ** (1 / holding_years)) - 1.
</description>
<example>
MoM = exit_equity / sponsor_equity
IRR_approx = MoM ** (1 / n_years) - 1
</sample>
</memory_item>
